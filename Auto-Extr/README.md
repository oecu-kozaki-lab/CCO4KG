言語条件
・出力はすべて日本語

意味構文の条件
・SPOは「"S"の"P"は"O"である」と読めるようにする
・Sに「原因」「影響」「影響予測」「対策法」は使用しない

オントロジー構築の条件
・クラス階層（subClassOf）や分類は構築しない
・「何によって事象が起こるのか」が単体の概念で分からない場合、必ず-を用いて「事象の対象-事象」形式で分離する
例：
・漁獲量増加→サンマ-漁獲量増加
・枯死→稲-枯死

因果方向の条件
・SPOは、"SのPはOである"を満たす
・間接的な関係の場合は、中間概念を介して直接的な関係で表現すること（中間概念も資料から抽出）。
例：
・大阪府　対策　囲いわな→大阪府　対策　害獣対策、害獣対策　対策手法　囲いわな

SPOの制約条件
・「原因」「影響」「影響予測」「対策法」は述語として使用
・「述語を指定しない」場合は、「原因」「影響」「影響予測」「対策法」以外を使用しても良い（その場合は述語を名詞化してください）

上記の条件を必ず守り、以下の（1）～（4）の手順に従って構築してください
(1) 主題トピックを1つ設定し、以下の4項目を文書で整理
    ・原因
    ・影響
    ・影響予測
    ・対策法

(2) (1)の文書を基に、以下の述語を使ったSPOを可能な限り構築
    ・原因
    ・影響
    ・影響予測
    ・対策法

(3) (2)で出た「目的語」を主語にして、資料を基にSPOを可能な限り構築（述語を指定しない）
  
(4) (3)の「目的語」を主語にして資料を基にSPOを可能な限り構築（述語を指定しない） 

(2)、(3)、(4)の出力形式
・S　P　O
